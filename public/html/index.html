Introduction


In this Core java Tutorials is specailly designed for Student and Working professional. Java is a simple, secure,robust, multithred, dynamic, light-weight, High Performance, Distributed, Object Oriented platform, independent , open source, Evergreen Programming;

since java can be used to develop all kinds of Applicatons like a Games , Mobile Application, website, Networking Application etc. So that we can say java is a Programming suite.

Java was developed by Sun Microsystems (which is now the subsidiary of Oracle) in the year 1995. James Gosling is known as the father of Java. Before Java, its name was Oak. Since Oak was already a registered company, so James Gosling and his team changed the name from Oak to Java.



There are Three Modules of Java


1. JSE / J2SE  (Java Standard Edition / Java 2 Platform Standard Edition)

It is a base module of remaining two modules (JEE or JME). It is a installable software as JDK.This Mudule is Given to Develop "Standalone Application". It provides basic java Libraries ad concepts like OOPS, Exception Handling, Threadinng Mechanism, Collections etc. These things will be used even when you are writing a Web-Application (J2EE). We use JSE to develop Standalone Application. It includes core topics like OOPs, String

About Standalone Application:

Standalone applications are also known as desktop applications or window-based applications. These are traditional software that we need to install on every machine. Examples of standalone application are Media player, antivirus, etc. AWT and Swing are used in Java for creating standalone applications.

2. JEE / J2EE  (Java Enterprise Edition / Java 2 Platform Enterprise Edition)

JEE moudels is not a installable software module. it is a set of specification (rules and guidelines) given by "Sun People" (SUNMS) to "third Party Companines or Vendors Companies" to develop the server software. It can be a "Web Server Software" or "Application Server Software" and being a Java Developer we are responsible to download and install the Server Software (like as TOMCAT , IIS, Apache web Server etc) and Work on that Software. Working on that software is nothing but "Working with on JEE Modules." We use JEE to develop Enterprise Application It includes topics like Servlet, JSP, Web Services, EJB, JPA

About Enterprise Application:

An application that is distributed in nature, such as banking applications, etc. is called an enterprise application. It has advantages like high-level security, load balancing, and clustering. In Java, EJB is used for creating enterprise applications.

3. JME / J2ME  (Java Micro Edition / Java 2 Platform Micro Edition)
It is a installable software as MDK (Mobile Development Kit). We can develop embeded system application with the help of JME Module. It is useful to develop Micro Application (Mobile Application).

About Mobile Application:

An application that is distributed in nature, such as banking applications, etc. is called an enterprise application. It has advantages like high-level security, load balancing, and clustering. In Java, EJB is used for creating enterprise applications.
